# Adrian Wahyu Septianto Portfolio

Editorial-brutalist developer portfolio for Adrian Wahyu Septianto, built with Next.js for Vercel deployment and SEO.

## Tech

- Next.js App Router
- React
- TypeScript
- CSS-only editorial UI system
- SEO metadata, Open Graph preview, robots, sitemap, and JSON-LD Person schema

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production Build

```bash
npm run build
npm run start
```

## Vercel

1. Push this folder to a GitHub repository.
2. Import the repository in Vercel.
3. Add `NEXT_PUBLIC_SITE_URL` with the final production URL.
4. Deploy with the default Next.js settings.

## GitHub Notes

The `Project/` folder contains real working project archives and should not be published as-is. This portfolio only exposes safe demo summaries and dummy data extracted from the project structure.
